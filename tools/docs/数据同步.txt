每5分钟将public_admin 向 question库同步
原理:
1. 每张表记录updateTime更新时间，每条数据修改后更新updateTime时间。
2. 数据删除将status修改成-1， 1个月后根据updateTime将1个月之前的数据delete删除。
3. public_admin 向 question 时，首先获取上一次同步记录，获取最近一次同步时间节点$lastSynchronizationTime
4. 获取public_admin库最近5分钟内有修改的数据。如果status==1 ，删除question表中的数据。如果status != 0 进行更新或者写入。
